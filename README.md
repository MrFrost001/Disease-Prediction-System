# 📌 Project Overview

Disease Prediction System is an AI and Machine Learning-based web application that predicts possible diseases based on symptoms entered by the user.

The system uses a trained machine learning model to analyze symptom patterns and provide predictions. It also stores prediction history for future reference.

The purpose of this project is to provide an easy-to-use healthcare support tool that can assist users in identifying possible health conditions based on symptoms.

---

# 🎯 Objectives

The main objectives of this project are:

- Predict diseases based on user symptoms
- Reduce manual health assessment effort
- Store prediction history
- Provide a simple and user-friendly healthcare interface
- Demonstrate practical use of Machine Learning in healthcare

---

# 🚀 Features

## 🔍 Disease Prediction

Users enter symptoms through a web interface.

Example symptoms:

- Fever
- Headache
- Nausea
- Vomiting
- Fatigue
- Joint Pain
- Skin Rash
- Cough
- Weight Loss
- Yellow Eyes

The machine learning model analyzes the symptoms and predicts possible disease outcomes.

---

## 💾 Prediction History

The application stores previous predictions.

Features:

- Saves symptoms
- Saves predicted disease
- Stores historical records
- Allows users to review previous results

---

## 🧠 Machine Learning Integration

The application uses:

- Trained ML model (`model.pkl`)
- Label encoder (`label_encoder.pkl`)

The model:

- Receives symptom data
- Processes patterns
- Generates prediction results
